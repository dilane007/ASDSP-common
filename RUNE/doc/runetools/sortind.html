sortind
 
Description

ind=sortind(varargin)

As sort but keeps only index. Calculates the index along dimension dim, otherwise along the first non-singleton dimension.
 
Input
xany matrix
dimAn integer > 0 indicating the dimension along which the operation will be performed. Inputs are the same as to sort.
 
Output
indIndex along the dimension of the operation to the sort value.
 
Try 
sortind([2.3; 3.2]), sortind(rand(1,2,3),[],2)
 
See Also sort, max, min, minind
