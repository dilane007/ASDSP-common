sizes
 
Description

sm=sizes(varargin)

Sizes of matrices collected in a matrix.
 
Input
aa matrix of any type
bas above
cas above
 
Output
sma matrix with sizes of one matrix per row
 
Try 
sizes(ones(2,3,4),ones(1,2),ones(5,6,7)) => [2 3 4; 1 2 1; 5 6 7]
 
See Also sizem, maxsize, size
 
Source Code
function sm = sizes(varargin) % size for many matrices as a matrix

% by Magnus Almgren 970529 revised 030930
for j = 1:nargin
 siz = size(varargin{j});
 s{j} = siz;
 n(j) = length(siz);
end
sm = ones(nargin,max(n));
for j = 1:nargin
 sm(j,1:n(j)) = s{j};
end
