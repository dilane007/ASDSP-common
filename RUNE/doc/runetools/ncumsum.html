ncumsum
 
Description

s=ncumsum(varargin)

As cumsum but normalised along dimension dim. The last element of the output will be one.
 
Input
many numeric matrix
dimthe dimension along which the accumulated summation takes place
 
Output
sa cumulative normalised sum of m
 
Try 
ncumsum(rand(4,2,3),3)
 
See Also cumsum, sum, cumprod
 
Source Code
function s = ncumsum(varargin)  % normalised cumsum 

% by Magnus Almgren 990223

s = mdiv(cumsum(varargin{:}),sum(varargin{:}));
