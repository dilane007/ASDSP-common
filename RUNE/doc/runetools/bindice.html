bindice
 
Description

res=bindice(prob,dim)

Creates a binary matirx of the same size as prob. One of the element is set according to the probabilities along dim.
 
Input
probMatrix of any size with probabilities.
dimDimension along which the realisation is made
 
Output
resA binary matrix of the same size as prob.
 
Try 
bindice(ones(6,5)), bindice(10*rand(4,3,2),2)
 
See Also binmin, binmax, ncumsum, ismin, mplus
 
Source Code
function res = bindice(prob,dim)

% by Magnus Almgren 000321 revised 030321 MA
if ~exist('dim','var')
 dim = firstnonsing(prob);
end
siz = size(prob);
siz(dim) = 1;
res = ismin(mplus(ncumsum(prob,dim),-rand(siz))<0,dim);
