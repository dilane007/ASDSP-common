norm1
 
Description

n=norm1(v,p,dim)

The vector norm (see norm) in dimension dim for any matrix v v and p must match each other in terms of size. Singleton dimension will be expanded to match.
 
Input
vThe target for the norm. Any numeric matrix.
pThe norm exponent. Any nonzero noninf real matrix. Default 2.
dimThe dimesion along which the norm is taken. Any integer greater than 0. Default first nonsingleton dimension.
 
Output
nthe norm value. It takes the combined size of v and p
 
Try 
norm1([3 6; 4 8])     => [5, 10]
norm1([3 6; 4 8],1,2) => [9; 12]  i.e. sum of each row
 
See Also norm, ncumsum
 
Source Code
function n = norm1(v, p, dim)  % vector norm along any dimension

% by Magnus Almgren 031015

if ~exist('dim', 'var')
  dim = firstnonsing(v);  % default in first no singleton dimension
 end
 if ~exist('p', 'var')
  p = 2; % default quadratic norm
 end
 n = mexp(sum(mexp(v,p), dim),1./p);
