mprod
 
Description

s=mprod(varargin)

Multiplies (.*) variables of different sizes. Arguments are expanded to fit each other and then multiplied together element by element. Expansion is made on singleton dimensions to fit the size of other arguments.
 
Output
sThe product (.*) of the inputs
 
Try 
mprod(ones(2,1),ones(1,2)), size(mprod(ones(2,1),ones(2,3,4)))
 
See Also mplus, mdiv, mexp, adjsiz, adjsiza
 
Source Code
function s = mprod(varargin) % multiplies a number of argument

% by Magnus Almgren 990317

v = adjsiz(varargin{:});
s = v{1};
for i=2:length(v)
 s = s.*v{i};
end
