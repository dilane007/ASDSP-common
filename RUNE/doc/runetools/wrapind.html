wrapind
 
Description

indo=wrapind(indi,siz)

Wraps or folds indo back to the range 1:size in a cyclic fashion. wrapind([-1 0 1 2 3 4], 3) => [2 3 1 2 3 1] Both input arguments are rounded before usage.
 
Input
indiAny real matrix of any size.
sizA scalar value not exceeded of the output. It must be >=0.5.
 
Output
indothe folded version of indo. Has the same size as indi
 
Try 
wrapind([-1 0 1 2 3 4], 3)
 
See Also mod, floor, round, ceil
 
Source Code
function indo = wrapind(indi, siz)

% by Magnus Almgren 951104

indo = round( mod(indi-0.5, round(siz) ) + 0.5);
