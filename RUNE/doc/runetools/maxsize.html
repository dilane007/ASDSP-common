maxsize
 
Description

res=maxsize(varargin)

Takes any number of inputs and leaves the maximum size for all of them.
 
Input
a,b,can argument list with variables of different sizes
 
Output
resA vector indicating the maximum size needed in order to contain any of the inputs.
 
Try 
maxsize(ones(1,2,3), ones(2,1,3,4))
 
See Also sizem, sizes
 
Source Code
function res = maxsize(varargin)

% by Magnus Almgren 970527 revised 031118

res = max(sizes(varargin{:}),[],1);
