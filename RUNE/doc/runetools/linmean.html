linmean
 
Description

y=linmean(varargin)

The same as mean but the input argument and the outpout are in dB. y = lin2db(mean(db2lin(arg),dim)). dim is optional.
 
Input
mA matrix with real values representing entities in dB.
dimThe dimension in which the mean value is computed
 
Output
yThe corresponding matrix cumsummed in the linear domain.
 
Try 
linmean([0; 0]),linmean([0, -inf])
 
See Also cumsum, linsum, lincumsum
 
Source Code
function y = linmean(varargin)

% by Magnus Almgren 011022

% mean value in the linear domain and back to dB
y = lin2db(mean(db2lin(varargin{1}),varargin{2:end}));
