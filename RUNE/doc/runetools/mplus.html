mplus
 
Description

s=mplus(varargin)

Adds variables of different sizes. Arguments are expanded to fit each other and then added together. Expansion is made on singleton dimensions to fit the size of the other arguments.
 
Output
sSum of the expanded inputs.
 
Try 
mplus(ones(2,1),ones(1,2)), size(mplus(ones(2,1),ones(2,3,4)))
 
See Also mprod, mdiv, mexp, adjsiz, adjsiza
 
Source Code
function s = mplus(varargin) 

% by Magnus Almgren 990317

v = adjsiz(varargin{:});
s = v{1};
for i=2:length(v)
   s = s+v{i};
end
