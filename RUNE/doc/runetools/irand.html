irand
 
Description

res=irand(varargin)

Complex random rectangular distributed numbers in the real as well as in the imaginary part.
 
Output
resA matrix of size according to input with complex elements. The real part is in the range [0 1] and the imaginary within [0 i].
 
Try 
plot(irand(1000,1),'.')
 
See Also irandn, setseed
 
Source Code
function res = irand(varargin)

% by Magnus Almgren 970611

res = rand(varargin{:}) + i*rand(varargin{:});
