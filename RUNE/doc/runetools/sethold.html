sethold
 
Description

holdstate=sethold(state)

Controls hold on and off for plots. outputs the current holdstate. Then a new state is set. Without argument it leaves the corrent state
 
Input
statethe new state to be used 1 <=> hold on, 0 <=> hold off.
 
Output
holdstateThe state before change.
 
Try 
sethold(1), sethold(0), sethold
 
See Also plotnh, ploth, ishold, hold
 
Source Code
function holdstate = sethold(state)

% by Magnus Allmgren 010220
holdstate = ishold;
if nargin >0
 if state 
  hold('on')
 else
  hold('off')
 end
end
