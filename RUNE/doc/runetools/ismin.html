ismin
 
Description

bind=ismin(x,dim)

Calculates a binary matrix pointing to the minimum element in that dimension , otherwise along the first non-singleton dimension.
 
Input
xany matrix
dimthe dimension along which the operation will be performed.
 
Output
bindBinary matrix pointing to the min value in dimension dim.
 
Try 
ismin([2.3; 3.2]),ismin(rand(5,6),2)
 
See Also ismax, min, minind
 
Source Code
function bind = ismin(x,dim)

% by Magnus Almgren 030930
if ~exist('dim','var') % defauld dimension dim
 dim = firstnonsing(x); % first non singleton dimension
end
[dummy, ind] = min(x,[],dim); % get index to min values
bind = false(size(x)); % create a logical matrix with filled with "false" 
bind(index1(ind,dim,size(x))) = true; % set true on positions with min values
