nans
 
Description

n=nans(varargin)

Creates a NaN matrix of size (r,c,...).
 
Input
Of the same type as i.e. ones.
 
Output
nA matrix of size according to input containing only NaN values
 
Try 
nans(1,2,3) or nans([1 2 3])
 
See Also zeros, ones, infs
 
Source Code
function n = nans(varargin)

% by Magnus Almgren 981119

n = NaN + ones(varargin{:});
