minind
 
Description

ind=minind(varargin)

As min, but keeps only index. Calculates the index along dimension of the third argument, otherwise along the first non-singleton dimension.
 
Input
xany matrix
dimAn integer > 0 indicating the dimension along which the operation will be performed. Inputs are the same as to min.
 
Output
indIndex along the dimension of the operation to the min value.
 
Try 
minind([2.3; 3.2]), minind(rand(1,2,3),[],2)
 
See Also min, maxind
 
Source Code
