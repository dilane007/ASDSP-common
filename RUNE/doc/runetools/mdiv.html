mdiv
 
Description

s=mdiv(varargin)

Divides (./) variables of different sizes. No warning if divide by zero. Arguments are expanded to fit each other and then divided element by element. Expansion is made on singleton dimensions to fit the size of other arguments.
 
Output
sThe ratio (./) of the inputs
 
Try 
mdiv(ones(1,2),ones(2,1)), size(mdiv(ones(2,1),ones(2,3,4)))
Compare 0/0 with mdiv(0,0)
 
See Also mplus, mprod, mexp, adjsiz, adjsiza
