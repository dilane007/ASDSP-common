rotate
 
Description

y=rotate(x,rot,dim)

according to rot in dimension dim. If dim is not present then first dimension is assumed
 
Input
xthe matrix to be shifted
rota matrix containing the shift
dimthe dimension along which shift will take place
 
Output
ythe shifted result
 
Try 
rotate(1:10,(0:3)',2) rotate(magic(2),[0 1])
 
Source Code
function y = rotate(x, rot, dim) % shifts first argument 

% by Magnus Almgren 981114 heavily revised 040202

setifnotexist('dim',firstnonsing(x))
[x, rot] = adjsiza(x, rot);
siz = size(x);
d = wrapind(mplus(flatten(1:size(x,dim),dim),-rot),size(x,dim));
ind = index1(d,dim,siz);
y = x(ind);
