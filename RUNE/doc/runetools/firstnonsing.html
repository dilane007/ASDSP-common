firstnonsing
 
Description

dim=firstnonsing(varargin)

Finds first non singleton dimension in any of the input arguments if all arguments are scalars or empty then dim = 1;
 
Input
a,b,cany number of matrices of any kind
 
Output
dimThe first non singleton dimension in any of the matrices
 
Try 
firstnonsing(rand(1,1,2), rand(1,2,2)) => 2
 
See Also firstsing
 
Source Code
function dim = firstnonsing(varargin) % find first non singleton dimension

% by Magnus Almgren 031015

dims = find(maxsize(varargin{:})>1);
if isempty(dims)
 dim = 1;
else
 dim = dims(1);
end
