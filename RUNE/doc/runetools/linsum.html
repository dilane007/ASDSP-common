linsum
 
Description

y=linsum(varargin)

Takes the same argument list as sum.m, but calculates lin2db(sum(db2lin(m),dim)). dim is optional.
 
Input
mA matrix with real values representing entities in dB.
dimThe dimension in which the opration is performed default is 1.
 
Output
yThe corresponding matrix summed in the linear domain.
 
Try 
linsum([0; 0]), linsum([0 0],2), linsum([-inf 0])
 
See Also sum, linmean, lincumsum
 
Source Code
function y = linsum(varargin) 

% by Magnus Almgren 990226

% sum in the linear linear domain and back to dB
y = lin2db(sum(db2lin(varargin{1}),varargin{2:end}));
