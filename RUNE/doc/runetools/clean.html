clean
 
Description

res=clean(a)

Keeps only finite values of matrix a.
 
Input
aany matrix
 
Output
resA vector spanning the first dimension with all non finite elements removed. Non finite elements are NaN and Inf
 
Try 
clean([nan 3 nan 2 inf])
 
Source Code
function res = clean(a)

% by Magnus Almgren 951104 revised 040202

res = flatten(a(isfinite(a)));
