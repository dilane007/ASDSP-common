db2lin
 
Description

varargout=db2lin(varargin)

Converts dB to linear. Takes any number of arguments. -Inf will map to 0.
 
Input
x1Real, matrix
x2Real, matrix
x3Real, ... Any number of any real matrices.
 
Output
y1As 1st input, but coverted.
y2As 2nd input, but coverted.
y3As 3rd input, but ...
yThe input arguments converted.
 
Try 
db2lin(3), db2lin([0 -inf]), db2lin([0 0], -3+zeros(2,2,2))
 
See Also lin2db
