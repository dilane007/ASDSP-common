qfuncinv
 
Description

x=qfuncinv(y)

y = qfunc(x) The inverse Q function If the input is outsde {0..1} Nan is delivered
 
Input
yAny real matrix
 
Output
xQ^-1(x)
 
Try 
qfuncinv(qfunc([-inf -1 0 1 inf NaN]))
plot(qfuncinv(0:0.025:1),0:0.025:1,'.-')
plot(-2:0.1:2,qfuncinv(qfunc(-2:0.1:2)),'.-')
 
See Also erfc, qfunc
