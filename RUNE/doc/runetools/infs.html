infs
 
Description

n=infs(varargin)

Creates an Inf matrix of size siz.
 
Output
nA matrix of size according to input containing only Inf (Infinite values).
 
Try 
infs(1,2,3) or infs([1 2 3])
 
See Also zeros, ones, nans
 
Source Code
function n = infs(varargin)

% by Magnus Almgren 98111

n = Inf + ones(varargin{:});
