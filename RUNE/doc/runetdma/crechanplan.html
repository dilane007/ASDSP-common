crechanplan
 
Description

obk=crechanplan(nb,nk,ncluster);

Makes a "frequency" plan with cluster size nb/ncluster.^2. The cells are numbered in the same way as in crecells. To get a correct symmetric frequency plan (nk*ncluster.^2)/nb has to evaluate into an integer.
 
Input
nbNumber of base stations.
nkNumber of channels in the system.
nclusternumber of clusters.
 
Output
obkBinary matrix, bases by channels. obk(j,k)=1, if base j is an allocated channel.
 
Try 
spy(crechanplan(12,24,3))
 
See Also crecells
 
Source Code
function obk=crechanplan(nb, nk, ncluster);

cellpclust = ceil(nb/ncluster); % clustersize;
obk = repmat(eye(cellpclust),[ncluster, ceil(nk/cellpclust)]);
obk = obk(1:nb,1:nk); % Correct the size if
                      % (nk*ncluster)/nb was not an integer.
